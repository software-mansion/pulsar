# DM Sans Font Files

To complete the setup, you need to add the DM Sans font files to this directory.

## Download Instructions

1. Visit Google Fonts: https://fonts.google.com/specimen/DM+Sans
2. Download the following font files and place them in this directory:
   - DMSans-Regular.ttf
   - DMSans-Bold.ttf
   - DMSans-Italic.ttf
   - DMSans-BoldItalic.ttf

## Alternative: Download via curl

```bash
cd /Users/piaskowyk/projects/pulsar/PulsarApp/assets/fonts

# Regular
curl -o DMSans-Regular.ttf "https://fonts.gstatic.com/s/dmsans/v16/raxhHp0_dCvaTM-Z6YjL-RJQzc5xH3aBJz01Zg.ttf"

# Bold
curl -o DMSans-Bold.ttf "https://fonts.gstatic.com/s/dmsans/v16/raxiHp0_dCvaTM-Z6YjL-RJks_5EKxNL-Z0Zg.ttf"

# Italic
curl -o DMSans-Italic.ttf "https://fonts.gstatic.com/s/dmsans/v16/raxdHp0_dCvaTM-Z6YjL-RmG-_aJ0N8I8pUN.ttf"

# Bold Italic
curl -o DMSans-BoldItalic.ttf "https://fonts.gstatic.com/s/dmsans/v16/raxeHp0_dCvaTM-Z6YjL-RmG-7-3zOAOxp1oqw.ttf"
```

Once you've added the font files, DM Sans will be loaded as the default font throughout your app.

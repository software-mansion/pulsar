50 web-haptic patterns to try in the browser

Each card defines a HapticPattern built from continuous blocks and PWM pulse blocks. Patterns are compiled with the same logic as PatternComposer (20–200 ms shot/pause window) and shipped to navigator.vibrate. Open this page on a phone with Web Vibration support to actually feel them — desktop browsers will just show the compiled timing array in the log.
